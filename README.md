# Turtle Crossing (Frogger-style Game)

A Frogger-style game built with Python and the Turtle module, showcasing Object-Oriented Programming (OOP) principles.

## Features
- Player controls a turtle to cross moving lanes of cars
- Increasing difficulty with each level
- Score/level tracking
- Modular design: separate classes for `Player`, `CarManager`, and `Scoreboard`

## Requirements
- Python 3.x
